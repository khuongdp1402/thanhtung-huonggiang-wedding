# Thiệp Cưới Online

Next.js wedding invitation – App Router, Tailwind v3, Framer Motion.

## Chạy local

```bash
npm install
npm run dev
```

Mở http://localhost:3000

## Deploy Vercel

1. Đẩy code lên GitHub.
2. Vào [vercel.com](https://vercel.com) → Add New Project → chọn repo.
3. Framework: Next.js (tự nhận). Deploy.

## Cấu hình

- **Next.js 14** (App Router), **Tailwind v3**, **PostCSS** – build ổn định, không Lightning CSS.
- **public/** – ảnh, nhạc, xi-character, happiness-sparkle-icon.
- Chỉnh nội dung thiệp trong `lib/invitation.ts`.
